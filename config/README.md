# Configuration

This directory contains **example** configurations for development
and testing. They demonstrate the structure and schema of CARF
policy definitions.

Production-grade configurations -- including calibrated thresholds,
industry-specific CSL-Core constraint libraries, and pre-verified
OPA policy bundles for EU AI Act compliance -- are available under
commercial license.

## Files

- `policies.yaml` -- Guardian policy structure (example thresholds)
- `agents.yaml` -- Agent configurations
- `policies/` -- CSL-Core formal policy definitions (example rules)
- `federated_policies/` -- Domain-owner governance policies (examples)
- `governance_boards/` -- Compliance board templates
- `policy_scaffolds/` -- Domain-specific policy templates
